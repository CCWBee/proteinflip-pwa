# ProteinFlip

A daily protein tracker, installable as a Progressive Web App and happy offline. Log what you eat against a target, watch the number climb, and let it reset itself each morning. No account, no server, no fuss.

It started life as a native iOS app, in SwiftUI; the web version overtook it and is now the one that gets used. That original native build is kept in `ios/` as an archive, no longer developed but preserved for the record.

## Run

Open `index.html`, or add it to your home screen from a browser that speaks PWA.
